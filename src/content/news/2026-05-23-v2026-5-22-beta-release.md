OpenClaw has tagged v2026.5.22-beta.1, a documentation and clarity-focused release published May 23, 2026. The pre-release adds or refines docs covering README onboarding, Gateway startup paths, WhatsApp QR/408 recovery, cron output language prompts, skill advanced features, gateway upstream 403 troubleshooting, and plugin fallback override guidance.

Additional clarifications address context-pruning ratio bounds, local dashboard recovery, CLI env markers, remote onboarding token behavior, Peekabox Bridge permissions for subprocess agents, browser CDP diagnostics, Plugin SDK allowlist imports, status-reaction timing defaults, queue steering behavior, limited-tool troubleshooting, cron HEARTBEAT handling, Telegram multi-agent groups, Bitwarden SecretRef setup, and EasyRunner deployments.

The release also consolidates GLM under Z.AI, adds an Upstash Box install guide and Gateway exposure runbook, clarifies MEDIA directives, Copilot and Voyage setup, config path quoting, real behavior proof, and memory-file write guidance. A Crabbox/Testbox improvement runs clean sparse-checkout Testbox syncs from a temporary full checkout routing remote changed gates through Corepack pnpm.
