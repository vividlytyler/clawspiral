OpenClaw shipped **v2026.5.19-beta.1** on May 19th, bringing several notable changes across the board.

**Meme Maker Skill** — A new skill ships that handles curated template search, local SVG/PNG rendering, Imgflip hosted rendering, and Know Your Meme provenance links. Useful for community content and social workflows.

**Mac App Settings Redesign** — The macOS app gets a full Settings page overhaul with consistent card layouts, cached navigation, cleaner permissions/voice/skills/cron/exec/debug panes, and steadier native sidebar spacing.

**Node.js Minimum Raised to 22.19** — Pi packages update to 0.75.1 and the minimum supported Node.js line moves to 22.19.

**Docker/Podman Change** — A new `OPENCLAW_IMAGE_APT_PACKAGES` runtime-neutral image build arg is added, with `OPENCLAW_DOCKER_APT_PACKAGES` kept as a legacy fallback.

**Gateway Improvements** — Two PRs from **@samzong** improve startup logging overlap with plugin service and channel sidecars, reducing restart ready latency. ACPX startup probes and resource counts are now better attributed in restart traces.

**Skills CLI Update** — `openclaw skills install` and `openclaw skills update` now support `--global` for targeting shared managed skills (PR #74466, thanks **@Marvae**).

**Other Fixes** — Plugin SDK/API deprecation paths are now explicit. The admin HTTP RPC plugin now allows trusted admin clients to start and wait for web QR login flows.

**Breaking** — If you're running Node.js below 22.19, you'll need to upgrade before updating.
