The OpenClaw team shipped version **2026.4.1** on April 1st, marking another substantial update to the open-source AI assistant platform. Here's what's new:

### `/tasks` — Native Background Task Board

The most user-facing change is `/tasks`, a chat-native background task board for the current session. It surfaces recent task details and agent-local fallback counts when no linked tasks are visible. Thanks to [@vincentkoc](https://github.com/vincentkoc) for the contribution (issue [#54226](https://github.com/openclaw/openclaw/issues/54226)).

### Bundled SearXNG Provider for Web Search

[@cgdusek](https://github.com/cgdusek) contributed a bundled SearXNG provider plugin for `web_search` with configurable host support ([PR #57317](https://github.com/openclaw/openclaw/pull/57317)). Self-hosters can now run their own meta-search engine and point OpenClaw at it — no API key required.

### Amazon Bedrock Guardrails

[@MikeORed](https://github.com/MikeORed) added Amazon Bedrock Guardrails support to the bundled provider ([PR #58588](https://github.com/openclaw/openclaw/pull/58588)). Enterprise users running models on AWS Bedrock can now apply content safety policies directly within OpenClaw.

### macOS Voice Wake

[@SmoothExec](https://github.com/SmoothExec) contributed Voice Wake for macOS — a new option to trigger Talk Mode via voice activation ([PR #58490](https://github.com/openclaw/openclaw/pull/58490)).

### Feishu Document Comments

[@wittam-01](https://github.com/wittam-01) built a dedicated Feishu Drive comment-event flow with in-thread replies and `feishu_drive` comment actions for document collaboration workflows ([PR #58497](https://github.com/openclaw/openclaw/pull/58497)).

### Gateway & Agent Improvements

- **Webchat history truncation** is now configurable via `gateway.webchat.chatHistoryMaxChars` and per-request `maxChars` ([PR #58900](https://github.com/openclaw/openclaw/pull/58900)).
- **Agent default params** are now settable globally via `agents.defaults.params` ([PR #58548](https://github.com/openclaw/openclaw/pull/58548), thanks [@lpender](https://github.com/lpender)).
- **Agent failover** received a significant overhaul: same-provider auth-profile retries are now capped before cross-provider fallback, with a new `auth.cooldowns.rateLimitedProfileRotations` knob ([PR #58707](https://github.com/openclaw/openclaw/pull/58707), thanks [@Forgely3D](https://github.com/Forgely3D)).
- **Cron tools allowlist**: `openclaw cron --tools` enables per-job tool allowlists for tighter security on scheduled tasks ([PR #58504](https://github.com/openclaw/openclaw/pull/58504), thanks [@andyk-ms](https://github.com/andyk-ms)).

This release continues OpenClaw's rapid iteration pace, with contributions across the full stack — from enterprise AWS integrations down to macOS-only quality-of-life features.
