OpenClaw 2026.6.8 is now stable, landing on June 16 after a week of beta testing across three pre-release candidates. The headline change is channel delivery quality for Telegram and WhatsApp.

## What's New in 2026.6.8

**Telegram improvements:**
- Structured text rendering: tables, lists, expandable blockquotes
- Intentional line breaks preserved
- CLI-backed replies for richer interactions

**WhatsApp improvements:**
- ACP bindings now honored properly

These changes follow the 2026.6.6 security-focused release (tighter fail-closed boundaries around transcript, sandbox, MCP, browser, channel, and exec-approval paths) and the 2026.6.7-beta.1 reliability release (Telegram polling/preview recovery, Slack transcript retention, and image source preservation).

## Upgrade

```bash
npm install -g @openclaw/cli@latest
# or
openclaw update
```

Check your version with `openclaw --version`. The full changelog is on [GitHub](https://github.com/openclaw/openclaw/releases).
