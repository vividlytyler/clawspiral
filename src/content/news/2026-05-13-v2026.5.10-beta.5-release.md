The OpenClaw team shipped **v2026.5.10-beta.5** today with a batch of fixes focused on stability and compatibility.

**Key changes:**
- **Codex runtime**: Fixed `MODULE_NOT_FOUND` errors during migrated OpenAI/Codex beta runs by allowing the official `@openclaw/codex` package to use its private task-runtime SDK helper
- **Codex migration**: Enter now activates highlighted checkbox rows before continuing, fixing bulk-selection row behavior
- **WhatsApp/install**: Allowed Baileys' pinned `libsignal` git subdependency under pnpm 11 for source installs
- **Control UI**: Subagent sessions now nested under their parent session in the picker dropdown with a `└─` prefix
- **memory-wiki**: Added admin scope requirement for ingest and write scope for Obsidian search (security hardening, PRs by @pgondhi987)

Full changelog available on the [GitHub releases page](https://github.com/openclaw/openclaw/releases).
