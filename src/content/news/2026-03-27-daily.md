## The Big Picture

OpenClaw just had one of its biggest weeks since inception. NVIDIA announced enterprise-grade security tooling at GTC 2026, a Shenzhen mass-installation event went viral across China, and a substantial version 2026.3.22 release shipped with ClawHub, improved sub-agents, and 30+ security patches.

---

## NVIDIA GTC 2026: NeMoClaw

At GTC 2026, NVIDIA announced **NeMoClaw** — an official enterprise stack built on top of OpenClaw. Key features:

- **One-command install** of NVIDIA Nemotron models and the new **NVIDIA OpenShell runtime**
- **Kernel-level sandboxing** for autonomous agent actions
- **Out-of-process policy enforcement** to prevent runaway agent behavior
- **Privacy-aware model routing** — keeps sensitive data on-premise

The move positions OpenClaw as enterprise-infrastructure alongside its roots as a personal assistant. NVIDIA's Jensen Huang called OpenClaw "the clearest signal yet that the agentic AI era has arrived for real." TechCrunch ran the headline: *"NVIDIA's version of OpenClaw could solve its biggest problem — security."*

This coincides with OpenClaw's transition to an independent foundation and the launch of DGX Spark (Dell Pro Max GB10, $4,756).

## Shenzhen: "Raise a Lobster" Goes Mainstream

On March 6–9, 2026, nearly **1,000 people queued outside Tencent's Shenzhen headquarters** for free OpenClaw installations. Tencent's cloud computing unit facilitated the event. Self-organized community gatherings in Shenzhen drew 500+ people each, multiple weekends in a row.

The term **"raise a lobster" (養龍蝦)** became slang for running a personal OpenClaw instance — a reference to the lobster tank imagery from OpenClaw's branding. MIT Technology Review covered the grassroots gold rush, noting that hustlers are already monetizing the craze through paid installation services and custom skills.

This is a notable cultural shift: AI agent software going from developer curiosity to mainstream consumer phenomenon, with Tencent actively facilitating adoption rather than building a competing product.

## OpenClaw v2026.3.22 — The "Fix What Hurts" Release

The March 22 release is described by power users as the most significant update since going full-time on Mac Mini. Highlights:

- **ClawHub Marketplace** — centralized plugin/skills marketplace, now the default install target. `clawhub search [query]` to find and install skills from a single CLI command
- **`/btw` side conversation command** — run secondary conversations without disrupting the main thread
- **Adjustable sub-agent thinking** — tune reasoning depth per sub-agent instead of globally
- **Multi-model sub-agents** — assign different models to different sub-agents in the same session
- **Session bloat management** — fixes that were "quietly killing performance right now"
- **30+ security patches** included

## March 25 GitHub Release (v2026.3.1-era features landing stable)

The March 25 GitHub release shipped several features to stable:

- **Gateway/OpenAI compatibility** — `/v1/models` and `/v1/embeddings` endpoints added, plus passthrough for explicit model overrides. Broadens RAG and client compatibility
- **Microsoft Teams SDK migration** — now uses the official Teams SDK with streaming replies, welcome cards, prompt starters, typing indicators, and native AI labeling
- **Skills one-click install** — bundled skills (coding-agent, gh-issues, openai-whisper-api, session-logs, tmux, trello, weather) now offer dependency installation automatically when requirements are missing
- **Control UI overhaul** — skill status tabs (All / Ready / Needs Setup / Disabled), click-to-detail dialogs with API key entry, frosted-backdrop markdown preview
- **Docker `--container` flag** — run `openclaw` commands inside a running container directly: `openclaw --container [name]`
- **Discord auto-threads** — optional LLM-generated thread names via `autoThreadName: "generated"`
- **Android node expansion** — `camera.list`, `device.permissions`, `device.health`, `notifications.actions` (open/dismiss/reply), plus motion sensors and calendar events
