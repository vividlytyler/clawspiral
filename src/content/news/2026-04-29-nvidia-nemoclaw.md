NVIDIA announced **NemoClaw** at GTC 2026 (March 16, San Jose), an open source reference stack purpose-built for the OpenClaw agent platform.

**What it does:** NemoClaw lets users install NVIDIA Nemotron models and the newly announced **NVIDIA OpenShell** runtime in a single command. OpenShell, part of the NVIDIA Agent Toolkit, adds privacy and security controls designed to make self-evolving autonomous AI agents more trustworthy and scalable.

**The goal:** Bring enterprise-grade guardrails to the open-source OpenClaw ecosystem without requiring a full infrastructure overhaul.

**Status:** NemoClaw is alpha software, available in early preview starting March 16, 2026. It is not production-ready yet.
