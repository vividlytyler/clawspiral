OpenClaw has released v2026.4.1-beta.1, a beta that adds a new `/tasks` command as a chat-native background task board alongside various subagent stability improvements.

## Key Features

- **`/tasks` command**: Adds a session-local task board showing recent task details and agent-local fallback counts when no linked tasks are visible — integrated directly into the chat interface
- **Subagent stagnation fixes**: Subagents that previously got stuck in long-running loops now have clearer recovery paths
- **Chrome CDP improvements**: Upgraded Chrome DevTools Protocol integration for more stable browser automation
- **Messaging platform fixes**: Resolved integration issues with Slack, Telegram, and Discord

The release was contributed by community developer @vincentkoc and is available on GitHub as a pre-release. The full release notes are available on the [OpenClaw releases page](https://github.com/openclaw/openclaw/releases).
