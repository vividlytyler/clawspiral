OpenClaw's April pre-release delivers one of its biggest feature drops yet: built-in `video_generate` and `music_generate` tools, now available out of the box.

Key additions in **2026.4.1-beta.1**:

- **Video generation** via a new built-in tool, integrated with configured providers
- **Music generation** with bundled Google Lyria and MiniMax providers, plus ComfyUI workflow support for async tracking and delivery
- **New providers**: Qwen, Fireworks AI, StepFun, MiniMax TTS, Ollama Web Search, and MiniMax Search integrations
- **Amazon Bedrock**: Added Mantle support plus automatic inference-profile discovery
- **ComfyUI plugin**: Bundled workflow media plugin for local and cloud ComfyUI instances
- **Multilingual Control UI**: Support for Simplified/Traditional Chinese, Brazilian Portuguese, German, Spanish, Japanese, Korean, French, Turkish, Indonesian, Polish, and Ukrainian
- **ClawHub**: Search, detail, and install flows now in the Skills panel
- **Config cleanup**: Legacy public config aliases removed in favor of canonical paths

The release also removes several deprecated config paths (`talk.voiceId`, `talk.apiKey`, `agents.*.sandbox.perSession`, `browser.ssrfPolicy.allowPrivateNetwork`, etc.) with migration support via `openclaw doctor --fix`.

**Full release notes**: [github.com/openclaw/openclaw/releases](https://github.com/openclaw/openclaw/releases)
