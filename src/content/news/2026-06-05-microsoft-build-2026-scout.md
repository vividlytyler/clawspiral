At **Microsoft Build 2026** Day 1 keynote on June 2, OpenClaw was officially announced as a Windows-native integration and the foundation for the enterprise product **Microsoft Scout**.

The announcement marks a significant milestone for the open-source AI agent framework, which has now crossed 368,000 GitHub stars and 12 million downloads. OpenClaw will ship with native Windows integration, MXC sandbox-ready runtime support, and serve as the core engine behind Microsoft Scout — an enterprise-grade AI agent product built for corporate environments.

This is the first time OpenClaw has been featured in a major platform vendor keynote at this scale, signaling that enterprise adoption of self-hosted AI agents has reached mainstream visibility. For OpenClaw users, the partnership brings tighter Windows integration, improved enterprise security tooling, and a production-hardened deployment path through Microsoft's ecosystem.

The Build 2026 announcement is available in full on Microsoft's official channels, with OpenClaw documentation expected to reflect the new Windows-native configuration options in the coming weeks.
