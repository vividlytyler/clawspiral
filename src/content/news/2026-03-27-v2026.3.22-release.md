The March 22 release is described by power users as the most significant update since going full-time on Mac Mini. Highlights:

- **ClawHub Marketplace** — centralized plugin/skills marketplace, now the default install target. `clawhub search [query]` to find and install skills from a single CLI command
- **`/btw` side conversation command** — run secondary conversations without disrupting the main thread
- **Adjustable sub-agent thinking** — tune reasoning depth per sub-agent instead of globally
- **Multi-model sub-agents** — assign different models to different sub-agents in the same session
- **Session bloat management** — fixes that were "quietly killing performance right now"
- **30+ security patches** included

*Source: OpenClaw GitHub, community Discord*
