OpenClaw v2026.7.1 landed on July 13, and it's the biggest release the project has shipped in months. The headline number alone tells the scale: **3,063 pull requests from 532 contributors** — a new record for the project.

## What's new

**Control UI overhaul** is the most visible change. Conversations are now easier to organize side by side, with live task tracking, clearer chat controls, better usage and cost views, and Gateway health status kept immediately accessible. The guided onboarding flow has also been rewritten: it now checks connections before saving them and preserves progress if setup is interrupted.

**Mobile apps got substantial love.** iOS, iPadOS, Android, and macOS each received wide-ranging updates across setup, navigation, chat, voice, permissions, localization, offline reading, queued sends, and connection recovery.

**Model and provider support expanded significantly:**
- GPT-5.6 compatibility improvements across OpenAI and Codex routes
- Tencent Hy3 gains a complete setup path
- Meta Model API adds Muse Spark 1.1
- Broader support for Claude, Ollama, ClawRouter, and LongCat

**Codex and coding agents grow closer.** The new `openclaw attach` command gives Claude Code temporary access to a selected session. Codex delegation and native subagents now return tracked results more reliably, and Copilot gains broader provider choices.

**Messaging channels all improved.** Telegram sees live progress, photos and documents, topics, commands, retries, and better setup. Slack gets threads, cards, progress indicators, identity handling, and duplicate prevention. Discord improves replies, attachments, voice sessions, and reconnect behavior. Apple Messages gets better reply threading, typing indicators, and routing.

**Gateway no longer crash-loops.** A long-standing pain point is fixed: repeatedly failing Gateways now leave a stable repair path instead of restarting forever.

**Remote browser control and workspace terminals** also advance: scheduled work can now wake only when something changes, selected signed-in browser tabs can pair remotely and save completed downloads safely, and guarded terminals are available across web, iOS, and Android.

The full release notes with every PR and contributor are on the [official docs](https://docs.openclaw.ai/releases/2026.7.1).
