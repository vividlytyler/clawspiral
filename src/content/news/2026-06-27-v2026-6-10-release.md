OpenClaw 2026.6.10 is out, bringing automatic fast mode that lets the agent handle short conversational turns quickly before returning to normal mode for longer work — without losing visible state. The release also includes more reliable provider routing, session identity, and trusted tool policies, plus improvements to provider setup, diagnostics, and transcript tooling.

Following closely, 2026.6.11-beta.1 adds Slack relay mode, native Mattermost `/oc_queue` support, and per-DM model overrides for easier channel automation. Beta testers also contributed richer operator workflows and broader agent tooling support.

The May 2026 release sweep (v2026.5.3 through v2026.5.5) had already addressed Discord package SecretRef contract resolution failures, macOS LaunchAgent upgrade issues, and a broader focus on faster agent turns, fewer dependencies, and a cleaner package shape.
