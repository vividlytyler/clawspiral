OpenClaw v2026.7.1 shipped on July 13, 2026, marking one of the most significant user-facing updates in the project's history.

**Control UI gets a full redesign.** The interface now works like a single browser workspace for conversations and parallel work. Sessions appear in resizable panes that persist after reload, chat titles generate automatically, and recent sessions are easier to pin, group, rename, fork, archive, and mark read. A denser Sessions page and live Tasks view aim to reduce tab-juggling.

**Mobile apps get major updates** on iOS, Android, and macOS, with clearer pairing flows and improved Gateway health visibility directly in the UI.

**Model and provider support expands** to include GPT-5.6 compatibility, Tencent Hy3, and Meta Muse Spark 1.1, alongside stronger Codex and connected coding-agent workflows.

**Messaging platforms see substantial updates** — Telegram, Slack, Discord, and Apple Messages each receive feature improvements.

Under the hood, Gateway crash loops, scheduled work, remote browser control, workspace terminals, sessions, and goals all receive stability and performance fixes.

The release is available now via `npm update -g openclaw` or through the GitHub releases page.
