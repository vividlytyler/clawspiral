OpenClaw [v2026.7.1-beta.2](https://github.com/openclaw/openclaw/releases/tag/v2026.7.1-beta.2) dropped July 5th with a dense set of highlights spanning model support, messaging platforms, and core tooling improvements.

## What's New

**GPT-5.6 support** — OpenClaw now recognizes the GPT-5.6 model family across catalog, capability, and runtime selection paths. Nemotron Super's 1M context window is also now available.

**External harness attachment** — `openclaw attach` launches an external harness against an existing Gateway session, making interactive Codex-style workflows easier to resume and inspect.

**Telegram Codex workflows** — Telegram users can now start Codex pairing with `/login`, steer active Codex runs, and recover final replies across transient API failures.

**Event-driven cron runs** — A new `on-exit` schedule kind wakes an agent when a watched command exits, while session-targeted runs can detach cleanly.

**ClawRouter** — A new bundled provider plugin delivers credential-scoped dynamic model discovery, OpenAI-compatible and native Anthropic/Gemini transports, and managed budget reporting across OpenClaw usage surfaces.

**iOS 26 refresh** — The native app adopts the iOS 26 visual system with clearer Chat, Talk, onboarding, and reconnect flows, plus expanded localization across Apple and Android surfaces.

**iMessage polls** — Built-in poll creation, reading, and voting for iMessage users.

**Mac local Gateway setup** — The macOS app can now install and start its local Gateway automatically.

**Control UI overhaul** — Session-first sidebar, compact context meter, warm light theme, reasoning-effort slider, streamlined composer, and slash-command picker.

The release also ships safer scoped conversations with per-conversation tool and access boundaries via capability profiles.
