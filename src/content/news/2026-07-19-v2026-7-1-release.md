OpenClaw **v2026.7.1**, released July 13, 2026, is the most substantial update of the summer — bringing a sweeping overhaul of the Control UI, major mobile app updates, and a wave of new model and provider integrations.

## Control UI Overhaul

The Control UI has been completely reworked to feel more like a single browser workspace. Key improvements:

- **Session panes** — Multiple sessions can sit in resizable, draggable panes that persist across reloads
- **Live Tasks view** — Inspect and cancel active background work without leaving the UI
- **Improved session management** — Pin, group, rename, fork, archive, and mark read; supports custom groups and shared read/unread status
- **Better composer** — Attachments, model selection, voice, run state, and message actions now stay clear and stable across mobile and desktop
- **Usage page** — Provider-reported plans, quotas, balances, and budgets visible alongside 7/30/90-day charts that show zero-activity days
- **Accessibility** — Keyboard and focus behavior, contrast, assistive labels, and touch targets more consistent across screen sizes; more languages supported

## Mobile Apps Updated

Official iOS, Android, and macOS apps received major updates alongside the Control UI overhaul, with improvements to pairing, voice, and mobile-first workflows.

## New Models and Providers

- **GPT-5.6** — Added as default for new setups, with `/think ultra` for Sol and Terra and max reasoning for Luna
- **Tencent Hy3** — New provider integration
- **Meta Muse Spark 1.1** — Added to the model roster
- **Featherless** — New provider
- **Claude Sonnet 5 and Mythos 5** — Additional Claude model options
- **ClawRouter** — New routing provider

## Codex and Coding Agent Workflows

Stronger Codex integration and connected coding-agent workflows round out the release, targeting developers using OpenClaw as their primary coding environment.

## Channel Updates

Telegram, Slack, Discord, and Apple Messages each received substantial updates in this release, alongside fixes for Gateway crash loops, scheduled work, remote browser control, workspace terminals, sessions, and goals.
